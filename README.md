# Oficina Mecânica - Esquema Conceitual

## 📌 Descrição
Este projeto faz parte da formação **SQL Database Specialist** da DIO.  
O objetivo é construir um **esquema conceitual** para o gerenciamento de uma oficina mecânica, representado por um **diagrama entidade-relacionamento (DER)**.

## 🛠️ Entidades Principais
- Clientes
- Veículos
- Ordem de Serviço (OS)
- Serviços
- Mecânicos
- Peças

## 🔗 Relacionamentos
- Cliente possui Veículo  
- Veículo gera Ordem de Serviço  
- Ordem de Serviço inclui Serviços  
- Serviço executado por Mecânico  
- Serviço utiliza Peças  

## 📂 Estrutura do Repositório
- `Oficina Mecânica.png` → Diagrama entidade-relacionamento (DER).  
- `README.md` → Documento explicativo do projeto.  

## 🚀 Objetivo
Este projeto demonstra a aplicação prática de **modelagem de dados** em um cenário real de oficina mecânica, servindo como base para criação de um banco de dados relacional.
